# Talks'n Slides

## Build

1. run in git root directory:
```sh
nix develop
checkout-submodules # this command is provided by the nix development shell
```

2. only necessary if using Qubes OS: set the locale: `export LC_ALL="C"`

3. run in specific presentation directory:
```sh
build # this command is provided by the nix development shell
```
